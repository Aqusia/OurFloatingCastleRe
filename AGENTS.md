# Repo Entry Guide

這個 repo 的可維護應用程式主體目前不在 repo 根目錄，而是在：

- `Documents/New project/`

## 接手順序

1. 進入 `Documents/New project/`
2. 閱讀 `Documents/New project/AGENTS.md`
3. 再讀 `Documents/New project/README.md` 與 `Documents/New project/docs/`

## 強制規則

只要修改 `Documents/New project/` 內的程式，就要同步檢查文件是否要更新。

至少要評估：

- `Documents/New project/docs/game-design.md`
- `Documents/New project/docs/repository-guide.md`
- `Documents/New project/docs/changelog.md`
