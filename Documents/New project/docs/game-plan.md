- 左側固定導覽：

  - 行動
  - 戰鬥
  - 消息
  - 好友
  - 商店

- 新增通知、戰報、好友、簽到基礎功能

## 角色頁

目前角色頁已經能看到：

- 血量 HP
- 魔力 MP
- 精力
- 金錢
- 材料
- 攻擊 / 防禦 / 運氣
- 心態 / 韌性
- 背包
- 稱號
- 職業圖片預留區
- 狀態欄（Buff / Debuff）
- 裝備欄：
  - 武器
  - 副武器
  - 頭盔
  - 盔甲
  - 護膝
  - 寵物
  - 替身
- 賦能位 3 格
- 每日簽到
- 突擊簽到狀態顯示

## 行動系統

目前行動系統規則：

- 同一時間只會執行一件事
- 可以先排後續隊列
- 隊列總長度上限為一天
- 行動期間不能建立或加入房間
- 在線完成的行動獎勵 +20%

目前可排的行動：

- 訓練：
  - 釣魚
  - 跳繩
  - 伏地挺身
  - 讀書
- 恢復：
  - 休息
- 挖礦：
  - 淺層礦坑
  - 深層礦坑（等級 10 解鎖）

補充：

- 訓練會消耗精力
- 挖礦會消耗精力
- 休息不消耗精力

## 戰鬥系統

目前戰鬥流程：

- 建立房間
- 加入房間
- 房主開始討伐
- 自動戰鬥逐 tick 推進
- Boss 自動攻擊
- 戰鬥結束後寫入戰報

目前已修正：

- 建房 / 加房後首次狀態同步
- 戰鬥結束後 HP / MP 會回寫到角色主資料
- 戰鬥中不再停留在房間清單視圖

## 消息系統

目前有兩類：

- 戰報
- 通知

通知內容已包含：

- 每日簽到
- 行動完成
- 戰鬥結果
- 好友新增

## 好友系統

目前版本：

- 以 Email 加好友
- 可看到好友是否在線

後續可擴充：

- 房間邀請
- 私訊
- 陣營成員列表

## 商店

目前商店已有三類：

- 消耗品
- 基礎裝備
- 賦能道具

目前購買後會：

- 扣除金錢
- 依道具類型進背包或裝上對應欄位

## 資料存放

- 本地持久化：`server/data/store.json`
- Socket.IO：房間與戰鬥同步
- Supabase schema 草稿：`supabase/migrations/`
