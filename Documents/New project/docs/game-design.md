# 多人養成討伐原型進度

這份文件描述目前版本「實際已存在」的玩法、系統與頁面，不寫純想像中的未來規格。

## 何時更新這份文件

只要有以下任一變化，就要同步更新：

- 新增或刪除功能
- 修改遊戲規則、限制、獎勵
- 改變主要頁面內容
- 新增新的系統分頁

## 行動系統

目前規則：

- 同一時間只會執行一件事
- 可以先排後續隊列
- 隊列總長度上限為一天
- 行動期間介面與後端會鎖定建立 / 加入隊伍、角色配置、技能 / 秘籍、裝備、背包整理、商店 / 市場、協助工程、移動、探險、玩家攻擊與 Boss 挑戰等會改變角色狀態的操作
- 角色只要有任何隊列項目，就不算閒暇
- 在線完成的行動獎勵 +20%

目前狀態顯示：

- 閒暇中
- 訓練中
- 挖礦中
- 休息中
- 鍛造中
- 狩獵中

進攻、狩獵這類隊伍行動必須由隊長發起，而且隊伍內所有成員都必須是閒暇中。角色移動中也視為忙碌，不能再加入訓練、挖礦、鍛造、戰鬥、攻城、建設、修建，也不能調整角色配置、裝備、技能、背包排序或市場交易。

目前可排的行動：

- 訓練：
  - 釣魚
  - 跳繩
  - 讀書
  - 伏地挺身
  - 沉思
  - 拳擊
- 恢復：
  - 休息
- 挖礦：
  - 淺層礦坑
  - 深層礦坑

## 戰鬥系統

目前流程：

- 建立房間
- 加入房間
- 房主開始討伐
- 可隨時組隊，但開始討伐時會檢查全隊是否閒暇
- 自動戰鬥逐 tick 推進
- Boss 自動攻擊
- 玩家與 Boss 會依運氣、技巧和戰鬥狀態觸發特殊事件
- 已裝備的次要角色會在玩家行動後自動判定技能，不需要手動選招；觸發後會進入短冷卻
- 戰鬥結束後寫入戰報
- HP / MP / 精力結果會回寫到角色資料
- 戰鬥紀錄會標示特殊事件、Boss 攻勢與獎勵 / 勝利結果，方便快速讀戰報
- 戰鬥頁上方的 `BOSS COMMAND` 會集中顯示目前三種主要攻打入口：公會爬塔 Boss、世界 Boss、隊伍 Boss；每張卡會顯示 FUN 評分、目前節奏提示與可否開戰，角色忙碌時按鈕維持不可操作
- 隊伍 Boss 已接入 Admin `roomBossRules`：可調 Boss 名稱、回合間隔、HP / 攻擊倍率、勝敗戰鬥經驗、本能經驗與金幣獎勵

目前支援：

- 一般討伐房
- 個人戰鬥：在戰鬥頁的場景挑戰選擇場景刷怪，難度由場景用途與外層自動決定，不再手動選簡單 / 普通 / 困難 / 菁英
- 場景挑戰卡片支援橫向拖曳瀏覽，畫面會顯示場景難度、用途、外層與預估收益
- 探險會依場景產生 3-6 步遭遇：採集點偏低壓獸群與補給，礦區偏晶化礦獸與坍方壓力，商路偏盜匪 / 伏擊，公會 Boss 點偏菁英守衛
- 同地點玩家遭遇：戰鬥頁會列出目前同據點玩家；角色閒暇、HP / 精力足夠且不同陣營時才能攻擊。同陣營玩家會顯示但禁止互打。
- 玩家遭遇節奏已接入 Admin `playerAttackRules`：可調發起精力、最多回合、攻守勝敗經驗、基礎繳獲金幣、戰鬥等級 / 運氣金幣加成與繳獲上下限。
- 公會一般戰鬥：在戰鬥頁選擇不同場景挑戰，依場景用途提供不同描述與收益方向
- 公會爬塔 Boss：在獨立爬塔頁推進與挑戰，畫面會顯示當層進度條、Boss 摘要與忙碌鎖定原因
- 世界 Boss 競賽：所有公會挑戰同一個事件 Boss；Boss 名稱、HP、攻擊、回合數、首殺獎勵、重複勝利獎與失敗參與獎已接入 Admin `worldBossRules`
- 攻城相關戰鬥紀錄
- 詳細戰報檢視
- 依職責進行簡易戰鬥邏輯分流
- 戰鬥畫面可收起，戰鬥仍會在背景同步，結束後不再強制蓋住頁面
- 個人戰鬥、同地點玩家遭遇、公會一般戰鬥、公會爬塔 Boss、攻城 Boss 結束後會切到戰鬥頁並開啟戰鬥紀錄

### SAO 式連擊鏈（2026-06-11 重做）

- 連擊上限＝戰鬥等級：16 級最多接出 16 連擊（硬上限 20）。
- 每一擊都有不同招式名：4 個職業各有 10 個普通招式與 2 個終結技，同一輪連擊內招式不重複。
- 接續判定：技巧與戰鬥等級決定下一擊接不接得上；技巧也降低落空率，落空會中斷連擊。
- 逐擊暴擊：運氣決定每一擊的暴擊率（上限 35%）與暴擊倍率（最高 ~2.1x）。
- 終結技：連擊達 4 段以上收尾時觸發，爆發傷害並壓低 Boss 下一輪攻勢。
- 連擊資源：第 2 擊起每擊消耗 1 點行動資源（法師吃 MP，其餘職業吃精力）；資源耗盡連擊中斷，所以體力 / 精神＝連擊續航。
- 戰報逐擊成行，前端以連擊（藍）/ 暴擊（橙）/ 終結技（紅）分色，房間戰鬥畫面有 COMBO 計數彈出與逐擊傷害浮動演出。

### 8 屬性戰鬥定位

- 攻擊：戰士 / 刺客單擊基準傷害主來源。
- 防禦：承受傷害的百分比減傷（遞減收益公式 def/(def+80)）。
- 運氣：逐擊暴擊率與暴擊倍率、戰鬥金幣繳獲加成。
- 智慧：法師單擊傷害主來源、精準破防觸發加成、次要角色技能傷害。
- 體力：HP 與精力池上限、每回合精力回復（連擊續航）。
- 精神：補師單擊傷害主來源、MP 池、每回合回魔、治療量。
- 技巧：連擊接續率、落空率降低、全職業小幅傷害加成。
- 韌性：固定減傷、Boss 反制壓力抗性、裝備耐久損耗降低。

### 成長曲線

- 升級需求：`70 + level^1.5 × 40`（L1≈110、L5≈520、L10≈1340、L16≈2630、L20≈3650），前期快、後期放緩。
- 敵方縮放：探險 / 公會 Boss / 房間 Boss 的 HP 與攻擊依（平均）戰鬥等級成長，HP 帶二次項避免高等級秒殺；獎勵金幣與經驗同步縮放。
- 模擬驗證：各等級平均擊殺回合數穩定在 3–4 回合，最大連擊數＝等級，終結率隨等級從 0% 升到 ~35%。

特殊事件：

- 連擊鏈 / 終結技：見上方 SAO 式連擊鏈
- 精準破防：依技巧與智慧觸發，降低 Boss 下一次攻勢
- 隊友支援：低血隊友可被額外支援
- 危急閃避：低血時依運氣減傷
- Boss 反制：Boss 偶爾施加全隊壓力（韌性減免）
- 次要角色自動技能：依角色等級、主定位適配性與偏好裝備判定，會寫入 battle log 與 specialEvents

## 陣營系統

目前已有：

- 陣營加入
- 放射狀城市規劃地圖（2026-06-11）：世界中心向外分 5 個陣營扇區，首都在內環、外層一～四沿主幹道向外，虛線同心環道呈現「王城—關隘—邊境」層次
- 三國策略遊戲風格的 2D 城池戰略地圖，不是欄位卡片列表；所有陣營共用同一張大地圖，以勢力色塊呈現領土，城池以旗標節點散佈在地圖上，並用道路線連接核心、關隘與邊境據點
- 城池節點以小型城池圖示呈現，只有首都、目前位置、目的地與選中城池顯示標籤；點選城池後，右側詳情面板才顯示城防、設施、Boss 與移動 / 攻城 / 建設 / 修建操作
- 地圖支援縮放與捲動，避免城池資訊擠在同一個視窗內看不清楚
- 每個地圖節點都有用途：核心、採集、野外刷怪、公會 Boss、礦脈、商路
- 地圖節點會同時顯示用途標籤與特性標籤，讓採集點、刷怪點、公會 Boss 點、礦點與商路點更容易辨識
- 合作外交
- 宣戰
- 公庫科技升級
- 攻城
- 防守 / 修建城池
- 陣營工程協力與退出
- 玩家市場可見性與陣營關聯

層次規則：

- 每個玩家加入陣營後會位在該陣營的某個據點，目前預設在首都 / 核心據點。
- 每個陣營有核心據點與外層一到外層四，層次之間有距離。
- 每個據點都有特性、建設槽與設施列表。
- 核心據點偏管理、科技、倉庫與城防；外層一偏農業 / 採集；外層二偏 Boss 討伐與公會爬層；外層三偏礦脈與鍛造素材；外層四偏商路、市場與攻城前哨。
- 移動到自己陣營據點會寫入角色的移動狀態；移動不放進個人行動隊列，但判定為忙碌行動，抵達前不能執行其他角色行動、角色配置、裝備 / 市場操作或城池工程。
- 移動中必須清楚顯示出發地、目的地與抵達時間；地圖節點要標示目前位置與目的地。
- 位在自己的目前城池時不再提供「城內移動」操作，避免和城池間移動混淆；目前城池只保留建設、修建等操作。
- 在目前所在的自己陣營據點可以發起建設工程，完成後把設施加入該據點。
- 在目前所在的自己陣營據點可以發起防守 / 修建工程，完成後提高城防。
- 建設與修建會先扣陣營公庫金幣。
- 陣營成員可加入進行中的工程加速，也可以隨時退出；若所有人退出，工程取消並退回公庫成本。
- 公庫不發放給個人，也不使用素材；公庫金幣用於建設 / 修建與公會科技升級。
- 公會科技由領袖 / admin 升級，目前包含城堡、防禦、攻擊、支援、進攻速度。
- 陣營地圖只保留城池、移動、建設、修建、攻城 Boss 與公會爬塔 Boss；一般魔物與公會一般戰鬥入口在戰鬥頁。
- 陣營地圖正中心有「場景挑戰」入口，點擊後切到戰鬥頁選擇個人或公會戰鬥。
- 個人戰鬥與公會一般戰鬥不會進入個人行動隊列。
- 公會爬塔已獨立成「爬塔」頁，主要流程是先移動到公會 Boss 據點，再用「趕路」或「攻擊」模式推進塔層。
- 公會爬層記錄目前層數、最高通關層、當層 Boss、Boss 攻擊、步數、所需步數、Boss 是否已遇到、最近事件、獎勵摘要與進度。
- 趕路模式較高機率前進，適合快速找當層 Boss；攻擊模式前進較慢，但較容易遇到巡邏小王並取得素材 / 金幣 / 戰鬥經驗。
- 當層步數達門檻後，每次推進有機率遇到該層 Boss；遇到後挑戰 Boss 區開啟，玩家只能挑戰或撤退，不能繼續推進。
- 打贏當層 Boss 會提高公會最高通關層、解鎖下一層、重置步數與 Boss 解鎖狀態，並提供個人與公庫獎勵。
- 公會爬塔要求角色人在公會 Boss 據點且閒暇中；移動、駐防、行動隊列中都不能推進、挑戰或撤退。
- 公會爬塔節奏已接入 Admin `towerRules`：可調基礎步數、每 5 層增加步數、趕路 / 攻擊精力、前進率、找王率、小王率、Boss HP / 攻擊倍率與獎勵倍率。
- 爬塔頁會顯示好玩度評分，依目前層數、進度、是否遇到 Boss、角色是否在據點且可操作估算，用來快速判斷目前是推進期、刷怪期或王戰期。

攻城規則（2026-06-11 更新重點）：

- 城牆耐久（fortification）＝城池血量；歸零才易主，前端以「城牆耐久」顯示。
- 自動砲臺獨立輸出：每輪對每名攻方造成壓制傷害，火力隨城牆耐久下降而減弱；無人駐守時砲臺獨力開火且傷害保底，低等級或精力耗盡的攻方會被磨退 / 磨死。
- 守軍被動維修：沒有攻城戰時城牆自動回復（基礎 2/小時，每名駐軍 +4/小時）；攻方撤退後守軍駐防即可逐步修回。


- 沒有隊伍時，角色必須閒暇中才能單人攻城
- 有隊伍時，只有隊長可以發起攻城
- 隊伍攻城前，所有隊員都必須閒暇中
- 隊伍攻城會記錄多名參與者並消耗參與者 HP / MP / 精力
- 守城 Boss 有技能、獎勵摘要與不同獎勵類型
- 攻城勝利會增加公庫金幣，參與者也會取得個人金幣與材料
- 攻城戰 v2 改為持久化長戰場：攻方發起後，攻守雙方可在期間加入，伺服器依時間補跑回合。
- 玩家可在目前所在的我方城池駐防；駐防期間角色視為忙碌，可離線參與守城。
- 每輪先計算守軍對戰，再計算城池自動防禦；沒有守軍時仍以自動防禦、城防、地形優勢與設施抵抗攻勢。
- 攻方打不穿時只造成輕微城防損耗，但仍持續消耗精力；精力或生命耗盡會退出戰場。
- 城防歸零才會易主；未歸零時只保留損耗，守方撐到時間結束即守城成功。
- 城池防禦參數包含地形優勢、自動防禦、駐防上限與攻城抗性，可由 Admin 調整。

屬性在攻城 / 守城中的定位：

- 攻擊：提高攻方破防與對守軍傷害。
- 防禦：降低承受傷害，提高守方存活。
- 運氣：提高暴擊、繳獲、低損耗機率。
- 智慧：提高戰術判定、攻城器械效率與法術型傷害。
- 體力：提高 HP、駐防持久與攻城精力消耗抗性。
- 精神：提高 MP、支援治療與守城士氣。
- 技巧：提高命中、連擊、破防與攻城器械操作。

## 2026-05-31 角色定位、次要角色與技能槽

- 主定位系統保留，現有定位為戰士、刺客、法師、補師。主定位決定基礎數值、預設 loadout 與主定位技能。
- 次要角色是 3 格角色卡系統，可選真實動漫/遊戲角色，例如桐人、亞絲娜、漩渦鳴人、五條悟、林克。
- 每個次要角色提供數值加成、專屬武器概念與可解鎖特殊技能；同一角色不可重複放入多格。
- 次要角色裝備後會自動參戰並獲得角色經驗，不需要排隊升級。Lv.1 / Lv.3 / Lv.5 會逐步解鎖技能，等級會提高觸發率、技能倍率與數值加成。
- 次要角色技能不再要求玩家手動放進特殊技能槽；已裝備角色會在戰鬥中依機率與單場冷卻自動施放，戰報以「【角色技能】」標示。
- 角色技能戰報使用多段演出文字；桐人「星爆氣流斬」會顯示 16 hit、分段傷害與終結斬總傷害。其他次要角色技能也會用多段 log 呈現，不再只有一行。
- 每個次要角色有主定位適配性與偏好裝備，實際加成會依角色等級、目前主定位與裝備部位重新計算。
- 角色保留 1 個特殊技能槽作舊資料相容與後續主定位 / 秘籍特殊技使用。
- 主職業新增職業熟練度，每個職業獨立保存等級、經驗與解鎖狀態；切換職業仍可行，但戰力、主職業技能與進階效果吃該職業自己的熟練度。

## 2026-05-31 秘籍與成就

- 秘籍沿用 inventory 的 `manual` 類別。玩家先學會秘籍，之後可在秘籍槽中裝備 Buff。

- 成就系統已有獨立頁面與角色資料欄位。初版提供「升到 5 等」樣板成就，後續可補正式成就與獎勵。

## 2026-06-01 戰鬥分類與戰報

- 戰鬥頁明確分為三類：探險、公會 Boss、世界 Boss 競賽。
- 探險取代一般刷野怪；第一版採即時多步結算，單次探險會產生 3-6 步事件，每步可能遇到野怪、菁英怪、陷阱、寶箱或休整點。
- 公會 Boss 保留公會爬層與討伐進度，但入口與戰報語意改為公會 Boss，不再用「公會一般戰鬥」混稱。
- 世界 Boss 競賽讓所有公會挑戰同一個事件 Boss；每個公會進場都面對滿血 Boss，第一個擊敗者取得主要公庫資源，其他挑戰取得參與獎。
- 新戰報必須逐行保存：開戰、每步/每回合、玩家攻擊、角色技能、Boss 反擊、結果與獎勵都各自成行。
- 舊版亂碼戰報無法還原原文，前端只顯示一次 legacy 提示，避免整個戰報視窗被重複提示洗版。
